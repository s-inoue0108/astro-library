# Markdown

## 特殊な記法

### コールアウト

```md:Info
> **info**
> text
> text
```

```md:Warning
> **warning**
> text
> text
```

```md:Hint
> **hint**
> text
> text
```

### 数式

```tex:Inline
$ \frac{a}{b} = c $
```

```tex:Display
$$ \frac{a}{b} = c $$
```
